# Trilha de PHP

Bem-vindo ao repositório da **Trilha de PHP**! 🚀

Este espaço foi criado para armazenar e documentar todas as atividades, exercícios e sistemas desenvolvidos durante a trilha de aprendizado em PHP.

## 📁 Estrutura do Repositório

O repositório está organizado da seguinte forma:

### 📝 Exercícios

A pasta `Exercicios/` contém uma série de exercícios práticos para aprimorar conhecimentos em PHP, cobrindo diferentes conceitos e técnicas da linguagem.

### 🛒 Carrinho de Compra

Um sistema completo de carrinho de compras desenvolvido em PHP, demonstrando a implementação de:

- Modelos e Serviços
- Manipulação de Enums
- Tratamento de Exceções
- Estrutura orientada a objetos

📄 Veja mais detalhes em [carrinhoDeCompra/README.md](carrinhoDeCompra/README.md)

### 💰 Lista de Desconto

Sistema de gerenciamento e aplicação de descontos com funcionalidades relacionadas a preços e promoções.

📄 Veja mais detalhes em [listaDeDesconto/README.md](listaDeDesconto/README.md)

## 🎯 Objetivo

Esta trilha tem como objetivo consolidar conhecimentos em:

- ✅ Conceitos fundamentais de PHP
- ✅ Programação Orientada a Objetos (POO)
- ✅ Manipulação de dados e estruturas
- ✅ Tratamento de exceções
- ✅ Desenvolvimento de sistemas práticos

## 🚀 Como Usar

1. Clone o repositório
2. Navegue até a pasta desejada
3. Execute os arquivos PHP:
   ```bash
   php nomedoarquivo.php
   ```

## 📚 Conteúdo

- **Exercícios**: Problemas práticos com diferentes níveis de dificuldade
- **Projetos**: Sistemas completos que integram múltiplos conceitos
- **Documentação**: READMEs específicos em cada pasta com mais detalhes
